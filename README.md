# IPhish
Author Ry4anCxeven
A beginner-friendly phishing awareness and cybersecurity training toolkit developed for educational purposes and authorized security research.

## Features

* Beginner-friendly interface
* Multiple demo templates
* Automated setup
* Localhost support
* Tunneling support
* Docker support
* Linux & Termux compatible
* Lightweight and easy to use
* Educational phishing simulation environment

## Installation

Clone the repository:

```bash id="n6sl3i"
git clone https://github.com/ryancxeven/Iphish.git
```

Move to the directory:

```bash id="bh6j2y"
cd Iphish
```

Run the tool:

```bash id="t4v0ol"
bash iphish.sh
```

## Installation (Termux)

```bash id="w2j0x1"
pkg update && pkg upgrade
pkg install git curl php
git clone https://github.com/ryancxeven/Iphish.git
cd Iphish
bash iphish.sh
```

## Docker Usage

Pull Docker image:

```bash id="v4o0nk"
docker pull ryancxeven/iphish
```

Run container:

```bash id="jzj6tb"
docker run --rm -it ryancxeven/iphish
```
